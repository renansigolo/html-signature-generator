# Signature Generator

A simple signature generator written in node.js

<img src="https://raw.githubusercontent.com/renansigolo/html-signature-generator/main/signature-example.png" width="900">

## Install

```console
$ npm i html-signature-generator
```

## Usage

```console
$ npm run build
```
